== Description ==

RapidText AI Content for WooCommerce is a powerful plugin that revolutionizes how you create product content for your WooCommerce store. Using advanced AI technology, generate high-quality product titles, descriptions, SEO metadata, FAQs, and translations with just one click.

**Key Features:**

* **1-Click Content Generation** - Generate complete product content instantly
* **Product Titles** - Create compelling, SEO-optimized product titles
* **Product Descriptions** - Generate detailed, engaging product descriptions
* **SEO Metadata** - Automatically create meta titles and descriptions
* **FAQ Generation** - Create relevant FAQs for your products
* **Multi-language Support** - Generate translations for international markets
* **Bulk Operations** - Process multiple products simultaneously
* **Advanced AI Models** - Powered by RapidText AI's cutting-edge technology

**Perfect for:**

* E-commerce store owners
* Digital marketers
* Content creators
* SEO specialists
* WooCommerce developers

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ai-content-for-woocommerce` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Make sure WooCommerce is installed and activated
4. Navigate to WooCommerce > RapidText AI to configure your settings
5. Get your API key from https://app.rapidtextai.com
6. Start generating AI-powered content for your products!
